# 2025 HAIC Web 
- 2025 HAIC의 주제인 게임 <Boxes and Dots>를 플레이 해볼 수 있는 웹페이지입니다.
- 대회 진행 도중 모델을 제출하신 뒤 얻을 수 있는 리플레이 텍스트를 입력하여, 게임의 진행 내역을 확인할 수 있습니다.
- 하나의 웹페이지로 두 명이서 게임을 직접 플레이해볼 수 있습니다.
